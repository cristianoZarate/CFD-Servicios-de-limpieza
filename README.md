# Proyecto CFD Servicios - DUOC UC

Este repositorio contiene la entrega del proyecto de software para la empresa **CFD Servicios**. El objetivo es modernizar la gestión de servicios mediante un sistema de agendamiento automatizado y un catálogo digital interactivo.

## ⚠️ Estructura del Repositorio
Siguiendo los requisitos docentes, el proyecto se divide en:

* **📂 Documentación:** Informes, UML, Wireframes, MER y Planificación (Gantt).
* **📂 Producto:** Código fuente, Scripts de Base de Datos (MySQL) y Librerías.
* **📂 Gestión:** Documentos de definición del proyecto e identificación del equipo.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** React (Desplegado en Vercel)
* **Backend:** Java Spring Boot (Desplegado en Railway)
* **Base de Datos:** MySQL
